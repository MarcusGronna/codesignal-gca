# Contains Duplicate

## Problem Description

Given an array of integers, determine if any value appears more than once in the array. Return true if at least one value appears more than once, and false if every element is distinct.

## Function Signature

```csharp
bool HasDuplicate(int[] nums)
```

## Examples

### Example 1
**Input:** `[1, 2, 3, 1]`  
**Output:** `true`  
**Explanation:** The value 1 appears twice in the array.

### Example 2
**Input:** `[1, 2, 3, 4]`  
**Output:** `false`  
**Explanation:** All elements are distinct.
