# Count Even Numbers

## Problem Description

Given an array of integers, count and return the number of even integers in the array.

## Function Signature

```csharp
int CountEvens(int[] numbers)
```

## Examples

### Example 1
**Input:** `[1, 2, 3, 4, 5, 6]`  
**Output:** `3`  
**Explanation:** The even numbers are 2, 4, and 6.

### Example 2
**Input:** `[1, 3, 5, 7]`  
**Output:** `0`  
**Explanation:** There are no even numbers in the array.
