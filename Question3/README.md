# Sum of Positive Numbers

## Problem Description

Given an array of integers, calculate and return the sum of all positive integers in the array. Zero is not considered positive.

## Function Signature

```csharp
int SumPositives(int[] values)
```

## Examples

### Example 1
**Input:** `[1, -2, 3, 4, -5]`  
**Output:** `8`  
**Explanation:** The positive numbers are 1, 3, and 4. Their sum is 1 + 3 + 4 = 8.

### Example 2
**Input:** `[-1, -2, -3]`  
**Output:** `0`  
**Explanation:** There are no positive numbers in the array.
