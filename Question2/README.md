# Reverse a String

## Problem Description

Given a string, return a new string with the characters in reverse order.

## Function Signature

```csharp
string ReverseString(string input)
```

## Examples

### Example 1
**Input:** `"hello"`  
**Output:** `"olleh"`  
**Explanation:** The string is reversed character by character.

### Example 2
**Input:** `"world"`  
**Output:** `"dlrow"`  
**Explanation:** Each character's position is reversed.
