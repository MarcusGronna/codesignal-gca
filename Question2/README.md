# Question 2: String Reverse

## Problem Description

Write a function that reverses a string.

## Function Signature

```csharp
public static string ReverseString(string input)
```

## Parameters

- `input`: A string to reverse

## Return Value

- Returns the reversed string
- Returns an empty string if input is null or empty

## Examples

```csharp
ReverseString("hello") // returns "olleh"
ReverseString("abc") // returns "cba"
ReverseString("") // returns ""
ReverseString("A") // returns "A"
```

## Constraints

- String length: 0 ≤ length ≤ 10,000
