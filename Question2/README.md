# Question 2: Reverse String

## Problem Description
Write a function that reverses a given string.

## Function Signature
```csharp
public string ReverseString(string input)
```

## Parameters
- `input`: A string to be reversed

## Returns
- A string that is the reverse of the input string

## Examples

### Example 1:
**Input:** `"hello"`  
**Output:** `"olleh"`

### Example 2:
**Input:** `"CodeSignal"`  
**Output:** `"langiSedoC"`

### Example 3:
**Input:** `""`  
**Output:** `""`  
**Explanation:** Empty string reversed is still empty

### Example 4:
**Input:** `"a"`  
**Output:** `"a"`  
**Explanation:** Single character remains the same

## Constraints
- 0 ≤ input length ≤ 1000
- Input contains only printable ASCII characters
